# DEX IDs

## Introduction

KyberSwap maintains a mapping of DEX IDs for each DEX that has been integrated with the KyberSwap Aggregator. This mapping enables integrators to easily filter the list of preferred DEXs for routing when querying the KyberSwap Aggregator via the `includedSources` and `excludedSources` params.&#x20;

{% hint style="info" %}
#### Supported DEXs by chains

Please refer to [Supported Exchanges And Networks](../../getting-started/supported-exchanges-and-networks.md) for the full list of supported DEXs on each chain.
{% endhint %}

For your convenience, the DEX IDs for each integrated DEX is listed below with their corresponding official DEX names as well as the link to their respective apps for avoidance of doubt.&#x20;
